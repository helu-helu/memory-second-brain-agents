# Unity Editor settings reference

Use the Unity Editor Preferences and **Project Settings**A broad collection of settings which allow you to configure how Physics, Audio, Networking, Graphics, Input and many other areas of your project behave. [More info](comp-ManagerGroup.html)  
See in [Glossary](Glossary.html#ProjectSettings) to configure how the Editor and your project behaves.

Preferences are personal settings stored on your local machine. Project Settings are shared settings that define how your project builds and runs, and are typically stored in **version control**A system for managing file changes. You can use Unity in conjunction with most common version control tools, including Perforce, Git, Mercurial and Unity Version Control (UVCS). [More info](VersionControl.html)  
See in [Glossary](Glossary.html#versioncontrol) for your entire team.
