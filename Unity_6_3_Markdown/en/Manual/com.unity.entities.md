# Entities

[com.unity.entities](https://docs.unity3d.com/Packages/com.unity.entities@1.4/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

The Entities package provides a modern Entity Component System (ECS) implementation with a basic set of systems and components made for Unity.

## Version information

### Released for Unity

Package version 1.4.7 is released for Unity Editor version 6000.3.
