# Searching in the Unity Editor

Use the search functionality in the Unity Editor to find and manage resources efficiently in your projects. You can search for assets, **GameObjects**The fundamental object in Unity scenes, which can represent characters, props, scenery, cameras, waypoints, and more. A GameObject’s functionality is defined by the Components attached to it. [More info](class-GameObject.html)  
See in [Glossary](Glossary.html#GameObject), components, settings, menu commands across different areas of the Editor.
