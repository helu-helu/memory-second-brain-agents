# 2D Aseprite Importer

[com.unity.2d.aseprite](https://docs.unity3d.com/Packages/com.unity.2d.aseprite@3.0/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

2D Aseprite Importer is a package which enables the import of .aseprite files from the Pixel Art tool Aseprite.

## Version information

### Released for Unity

Package version 3.0.2 is released for Unity Editor version 6000.3.
