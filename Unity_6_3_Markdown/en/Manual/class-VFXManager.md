# VFX Project Settings

The VFX tab in the **Project Settings**A broad collection of settings which allow you to configure how Physics, Audio, Networking, Graphics, Input and many other areas of your project behave. [More info](comp-ManagerGroup.html)  
See in [Glossary](Glossary.html#ProjectSettings) window controls settings relevant to the Visual Effect Graph.

This tab is only visible if you install the Visual Effect Graph package. For information on how to install the Visual Effect Graph, see the [packages documentation](upm-ui-install.html).

For information on the properties in this tab, see the Visual Effect Graph [package documentation](https://docs.unity3d.com/Packages/com.unity.visualeffectgraph@latest/index.html?subfolder=/manual/VisualEffectProjectSettings.html).
