# Mesh data

Resources on the data that a **mesh**The main graphics primitive of Unity. Meshes make up a large part of your 3D worlds. Unity supports triangulated or Quadrangulated polygon meshes. Nurbs, Nurms, Subdiv surfaces must be converted to polygons. [More info](mesh.html)  
See in [Glossary](Glossary.html#Mesh) contains, and how to view it in the Unity Editor.

| [Mesh topology data](mesh-topology-data.html) | Learn about mesh topology data, which describes the type of face that a mesh has. |
| [Mesh index data](mesh-index-data.html) | Learn about mesh index data, which describes how vertices combine to create the mesh’s surface, based on the topology. |
