# Facebook Instant Games setup process

Set up your Unity project to use the Facebook Instant Games platform and upload builds directly from the Unity Editor to Facebook Instant Games.

To set up your Unity project for Facebook Instant Games, complete the following tasks:

* [Install the Web module](#install-module).
* [Create a Facebook Instant Games build profile](#build-profile).
* [Register as a Facebook Developer](#register).

Before you get started, check the [technical limitations](webgl-technical-overview.html) and [browser compatibility](webgl-browsercompatibility.html) for the Web platform to make sure you’re aware of any limitations for developing a Unity application for this platform.

## Install the Web module

The Facebook Instant Games platform uses the basic structure from the Web module.

To install the Web module:

* At install time, refer to [Downloading and installing Editors and modules with the Unity Hub](https://docs.unity3d.com/hub/manual/InstallEditors.html).
* To an existing install, refer to [Add modules](https://docs.unity3d.com/hub/manual/AddModules.html).

## Create a Facebook Instant Games build profile

The Facebook Instant Games platform is accessible through a [build profile](build-profiles.html)A set of customizable configuration settings to use when creating a build for your target platform. [More info](build-profiles.html)  
See in [Glossary](Glossary.html#buildprofile) workflow.

To access the Facebook Instant Games platform, follow the instructions in [Create a build profile](create-build-profile.html).

## Register as a Facebook Developer

To upload your build to Facebook Instant Games, you need information from a Facebook Developer account.

[Register as a Facebook developer](https://www.facebook.com/fbgaminghome/developers/instant-games/get-started/step-1) and review [Facebook’s Instant Games documentation](https://developers.facebook.com/docs/games/build/instant-games) to make sure your build functions well on Facebook Instant Games.
