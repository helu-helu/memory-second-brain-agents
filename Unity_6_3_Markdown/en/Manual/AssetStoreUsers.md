# Add a user to a publisher profile

Multiple users can access a single publisher profile to publish packages on the **Asset Store**A growing library of free and commercial assets created by Unity and members of the community. Offers a wide variety of assets, from textures, models and animations to whole project examples, tutorials and Editor extensions. [More info](AssetStore.html)  
See in [Glossary](Glossary.html#AssetStore). This is useful when you work as part of a team to create Asset Store packages.

Any user you add to the a publisher profile gets access to all the information on that profile and can make the same changes as the creator of that profile.

To associate a user with your publisher profile:

1. Open the [Asset Store Publisher Portal](https://publisher.unity.com).
2. Select the **Users** tab.
3. Select **Add new user**.
4. Enter the email address you want to access to this publisher profile.
5. Select **Continue**.

## Remove a user from a publisher profile

To remove an user’s access to your publisher profile:

1. Open the [Asset Store Publisher Portal](https://publisher.unity.com).
2. Select the **Users** tab.
3. Select the delete (**X**) icon next to the email address you want to remove.
