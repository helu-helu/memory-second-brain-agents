# Push Notifications

[com.unity.services.push-notifications](https://docs.unity.com/push-notifications/) ![](../uploads/Main/iconRel.png)

## Description

This package adds support for Push Notifications to your game. It allows sending rich push notifications with images, and provides analytics on the number of received push notifications.

## Version information

### Released for Unity

Package version 4.0.2 is released for Unity Editor version 6000.3.
