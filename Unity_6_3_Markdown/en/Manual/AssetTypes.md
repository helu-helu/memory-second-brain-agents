# Types of assets

Unity supports a variety of asset types that you can import and use in your project. You can import assets from a variety of sources, and manage them in the Unity Editor. You can also group together configurations of assets into prefabs to reuse across your project.

| **Topic** | **Description** |
| --- | --- |
| **[Supported asset type reference](assets-supported-types.html)** | Reference of asset file formats that Unity can import in your projects. |
| **[Models](models.html)A 3D model representation of an object, such as a character, a building, or a piece of furniture. [More info](3D-formats.html) See in [Glossary](Glossary.html#model)** | Import and configure 3D models from external modeling software for use in your **scenes**A Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. [More info](CreatingScenes.html) See in [Glossary](Glossary.html#Scene). |
| **[Meshes](mesh.html)** | Work with **mesh**The main graphics primitive of Unity. Meshes make up a large part of your 3D worlds. Unity supports triangulated or Quadrangulated polygon meshes. Nurbs, Nurms, Subdiv surfaces must be converted to polygons. [More info](mesh.html) See in [Glossary](Glossary.html#Mesh) data that defines the shape and structure of **3D objects**A 3D GameObject such as a cube, terrain or ragdoll. [More info](GameObjects.html) See in [Glossary](Glossary.html#3DObject) in Unity. |
| **[Prefabs](Prefabs.html)An asset type that allows you to store a GameObject complete with components and properties. The prefab acts as a template from which you can create new object instances in the scene. [More info](Prefabs.html) See in [Glossary](Glossary.html#prefab)** | Create reusable **GameObject**The fundamental object in Unity scenes, which can represent characters, props, scenery, cameras, waypoints, and more. A GameObject’s functionality is defined by the Components attached to it. [More info](class-GameObject.html) See in [Glossary](Glossary.html#GameObject) templates that you can instantiate multiple times throughout your project. |
| **[Text assets](class-TextAsset.html)** | Import and use text files, JSON, XML, and other text-based data in your project. |
