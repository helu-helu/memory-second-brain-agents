# Navigating and managing the Unity Editor

Navigate the Unity Editor with keyboard shortcuts, search, and the Undo History window. You can also use the toolbar and status bar to manage and monitor your project.

| --- | --- |
| **[Keyboard shortcuts](ShortcutsManager.html)** | Use the **Shortcuts** window to assign, manage, and save keyboard shortcuts, including shortcuts for custom tools, **scripts**A piece of code that allows you to create your own Components, trigger game events, modify Component properties over time and respond to user input in any way you like. [More info](creating-scripts.html) See in [Glossary](Glossary.html#Scripts), and packages. |
| **[Undo History window](UndoWindow.html)** | Navigate through your recent actions to undo or redo multiple steps at once. |
| **[Searching in the Unity Editor](editor-searching.html)** | Find assets, **GameObjects**The fundamental object in Unity scenes, which can represent characters, props, scenery, cameras, waypoints, and more. A GameObject’s functionality is defined by the Components attached to it. [More info](class-GameObject.html) See in [Glossary](Glossary.html#GameObject), settings, and other resources using Unity’s search tools. |
| **[Save your work](Saving.html)** | Understand how Unity saves your **scenes**A Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. [More info](CreatingScenes.html) See in [Glossary](Glossary.html#Scene), assets, and **project settings**A broad collection of settings which allow you to configure how Physics, Audio, Networking, Graphics, Input and many other areas of your project behave. [More info](comp-ManagerGroup.html) See in [Glossary](Glossary.html#ProjectSettings). |
