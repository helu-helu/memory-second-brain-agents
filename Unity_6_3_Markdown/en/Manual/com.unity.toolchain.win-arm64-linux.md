# Toolchain Windows Arm64 Host for Linux Targets

[com.unity.toolchain.win-arm64-linux](https://docs.unity3d.com/Packages/com.unity.toolchain.win-arm64-linux@1.1/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

This package provides a cross-compilation toolchain for building Unity players targeting Linux and Embedded Linux platforms. It is designed to run on Arm64 Windows hosts and supports both x86\_64 and Arm64 player targets
