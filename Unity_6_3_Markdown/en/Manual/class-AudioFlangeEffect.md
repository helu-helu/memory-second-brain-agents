# Audio Flange Effect

The **Audio Flange Effect** is used to create the **audio effect**Any effect that can modify the output of Audio Mixer components, such as filtering frequency ranges of a sound or applying reverb. [More info](class-AudioEffectMixer.html)  
See in [Glossary](Glossary.html#AudioEffect) produced by mixing two identical signals together, one signal delayed by a small and gradually changing period, usually smaller than 20 milliseconds.

## Properties

![The AudioGroup Inspector displays the configurable properties of an Audio Flange Effect.](../uploads/Main/AudioFlangeEffect.png)


The AudioGroup Inspector displays the configurable properties of an Audio Flange Effect.

| ***Property:*** | ***Function:*** |
| --- | --- |
| **Drymix** | Percentage of original signal to pass to output from 0.0 to 100.0%. Default = 45%. |
| **Wetmix** | Percentage of flange signal to pass to output from 0.0 to 100.0%. Default = 55%. |
| **Depth** | Intensity of the flange effect from 0.01 to 1.0. Default = 1.0. |
| **Rate** | Speed of the flange effect from 0.1 to 20 Hz. Default = 10 Hz. |
