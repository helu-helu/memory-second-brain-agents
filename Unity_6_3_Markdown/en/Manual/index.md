# Unity 6.3 User Manual

[Unity 6.3](https://unity.com/download) is the latest long-term support (LTS) release of the next generation of the Unity Engine. It combines the latest technologies and tools to deliver high-quality, high-performance experiences for all supported platforms.

For more information about new features and how to upgrade your existing projects to this version, refer to the following:

* [New in Unity](WhatsNew.html)
* [Upgrade Unity](UpgradeGuides.html)

### [Visual effects](visual-effects.html)

Add and customize **particle**A small, simple image or mesh that is emitted by a particle system. A particle system can display and move particles in great numbers to represent a fluid or amorphous entity. The effect of all the particles together creates the impression of the complete entity, such as smoke. [More info](class-ParticleSystem.html)  
See in [Glossary](Glossary.html#particle), **lens flare**A component that simulates the effect of lights refracting inside a camera lens. Use a Lens Flare to represent very bright lights or add atmosphere to your scene. [More info](class-LensFlare.html)
