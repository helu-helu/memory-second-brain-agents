# Check analytics data for your UPM packages

**Note**: Paid **UPM packages**A **Package** managed by the **Unity Package Manager**. Refer to **Packages**.  
See in [Glossary](Glossary.html#UPMpackage) are not yet available on the Publisher Portal. You can publish free UPM packages.

To learn how many users interact with your UPM packages, go to the ****Analytics**Abbreviation of **Unity Analytics**  
See in [Glossary](Glossary.html#Analytics)** section of the Publisher Portal.

![The Publisher Portal analytics window in the adoption tab view.](../uploads/Main/publisher-portal-analytics.png)


The Publisher Portal analytics window in the adoption tab view.

| **Analytics tab** | **Description** |
| --- | --- |
| **Reach** | The number of page visits to the UPM product details page (PDP). |
| **Adoption** | The number of times a user selected **Add to my Assets**. |
| **Usage** | The number of Unity Editor instances that have this package installed in the Package Manager. |

## Change the analytics graph view

The analytics graph contains the following chart views:

* **Line chart**
* **Stacked area**

To display product data in the graph, select the checkbox next to a product name. There’s no limit to the number of products you can display in the graph.

Select the date range dropdown to change the period of time displayed in the graph.

## Download usage data

The analytics .csv file contains reach, adoption, or usage data in text format as a series of strings.

To download analytics data for all projects in a specific tab:

1. Select an analytics tab that contains the data you want to download.
2. Select **Download all as CSV**.
