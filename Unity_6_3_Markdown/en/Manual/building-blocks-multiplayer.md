# Multiplayer Services Building Blocks

The Multiplayer Services Unity Building Blocks provide a UI-based path to quickly set up multiplayer sessions and matchmaking in your Unity project.

| **Topic** | **Description** |
| --- | --- |
| [Multiplayer Services Building Blocks prerequisites](building-blocks-multiplayer-prerequisites.html) | Set up your project to work with the [Unity Services](UnityServices.html) that the Unity Multiplayer Services Building Blocks use. |
| [Multiplayer Sessions Building Block](building-blocks-multiplayer-sessions.html) | Implement Unity’s multiplayer sessions in your project with this Building Block’s pre-made UI elements. |
| [Matchmaker Session Building Block](building-blocks-multiplayer-matchmaking.html) | Implement Unity’s multiplayer Matchmaker in your project with this Building Block’s pre-made UI elements. |
