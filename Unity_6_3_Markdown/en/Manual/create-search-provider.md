# Create a custom search provider

You can use the Search API to create your own [search provider](search-filter-by-provider.html). A custom search provider can combine search results from different sources, exclude properties from the search, and perform actions on the search results.

This section provides information on how to create a custom search provider.

| Topic | Description |
