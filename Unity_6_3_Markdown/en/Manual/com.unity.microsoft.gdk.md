# Microsoft GDK API

[com.unity.microsoft.gdk](https://docs.unity3d.com/Packages/com.unity.microsoft.gdk@1.6/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

Provides support for the Microsoft Game Development Kit making it available to approved platforms. Requires the Microsoft platform APIs to be pre-installed.
