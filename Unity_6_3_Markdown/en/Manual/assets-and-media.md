# Assets and media

An asset is any file you use in your project such as 3D models, textures, audio files, and **scripts**A piece of code that allows you to create your own Components, trigger game events, modify Component properties over time and respond to user input in any way you like. [More info](creating-scripts.html)  
See in [Glossary](Glossary.html#Scripts). Understand how to manage, import, and optimize assets, with Unity’s tools and workflows.
