# Gradle for Android

Gradle is a build system that automates build processes and prevents many common build errors. Unity uses **Gradle** for all Android builds. You can either build the output package .**apk**The Android Package format output by Unity. An APK is automatically deployed to your device when you select File > Build & Run. [More info](android-BuildProcess.html)  
See in [Glossary](Glossary.html#APK) or .aab file in Unity or export a Gradle project from Unity and build it with an external tool such as Android Studio.
