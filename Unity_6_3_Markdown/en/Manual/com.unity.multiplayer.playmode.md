# Multiplayer Play Mode

[com.unity.multiplayer.playmode](https://docs.unity3d.com/Packages/com.unity.multiplayer.playmode@2.0/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

Multiplayer Play Mode is a feature that enables multiple editor instances to be opened simultaneously on the same development device.
