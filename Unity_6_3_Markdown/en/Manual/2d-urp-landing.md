# 2D game development in URP

Create unique visual effects for your 2D URP project.

Use the different 2D URP features available to create the graphics and visual style you want in your 2D project. Simulate lighting in 2D **scenes**A Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. [More info](CreatingScenes.html)  
See in [Glossary](Glossary.html#Scene) with [2D lighting in URP](urp/2d-index.html), or create a pixelated visual style with the [2D Pixel Perfect Camera](urp/2d-pixelperfect.html). You can use [3D GameObjects in 2D URP scenes](urp/2d-renderer-urp-features-landing.html) to create unique visual effects with 3D meshes in your 2D project.
