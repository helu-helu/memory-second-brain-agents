# Entities Graphics

[com.unity.entities.graphics](https://docs.unity3d.com/Packages/com.unity.entities.graphics@1.4/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

The Entities Graphics package provides systems and components for drawing meshes using DOTS, including support for instanced mesh rendering and LOD.
