# Upload to Facebook Instant Games

Upload a build from the Unity Editor directly to Facebook Instant Games.

**Notes**:

* You need a [Facebook Developer account](https://www.facebook.com/fbgaminghome/developers/instant-games/get-started/step-1) to upload to Facebook Instant Games.
* Before uploading, review the upload guidance and hosting restrictions in Facebook’s documentation [Testing, Publishing and Sharing an Instant Game](https://developers.facebook.com/docs/games/build/instant-games/get-started/test-publish-share). Builds must meet these requirements to upload successfully.

To upload a build from the Unity Editor to Facebook Instant Games:

1. Go to **Edit** > **Build Profiles**.
2. Select your Facebook Instant Games **build profile**A set of customizable configuration settings to use when creating a build for your target platform. [More info](build-profiles.html)  
   See in [Glossary](Glossary.html#buildprofile).
3. Select **Upload**.
4. Enter the required information.
   **Note**: Find this information under **App settings** > **Basic** on your Facebook Developer account page.
5. Choose the build folder, which has the `index.html` file at its root.
6. Select **Upload**.

When your build uploads to Facebook Instant Games, it appears on the developer dashboard with any notes added in the upload process.
