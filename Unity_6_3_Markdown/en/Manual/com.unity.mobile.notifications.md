# Mobile Notifications

[com.unity.mobile.notifications](https://docs.unity3d.com/Packages/com.unity.mobile.notifications@2.4/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

Mobile Notifications package adds support for scheduling local repeatable or one-time notifications on iOS and Android.  
  
On iOS receiving of push notifications is also supported.

## Version information

### Released for Unity
