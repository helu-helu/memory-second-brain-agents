# Cubemaps

Resources for creating and using a collection of six square textures that represent the reflections in an environment, and form an imaginary cube around a **GameObject**The fundamental object in Unity scenes, which can represent characters, props, scenery, cameras, waypoints, and more. A GameObject’s functionality is defined by the Components attached to it. [More info](class-GameObject.html)
