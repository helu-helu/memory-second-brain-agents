# Deprecate an asset package on the Asset Store

Deprecation removes a published package from **Asset Store**A growing library of free and commercial assets created by Unity and members of the community. Offers a wide variety of assets, from textures, models and animations to whole project examples, tutorials and Editor extensions. [More info](AssetStore.html)  
See in [Glossary](Glossary.html#AssetStore) search results, but it still exists on the Asset Store. This tells the users that you no longer support this package. Any users that have acquired this package can still access it and download the latest version.

The Asset Store team can also deprecate packages on the Asset Store that don’t meet quality standards.

When you set a package as deprecated, the package no longer appears as a search result in the Asset Store. For new customers, it’s removed from the store. However, customers who have already purchased or downloaded the package can still download the package.

To deprecate a published package:

1. Open the [Asset Store Publisher Portal](https://publisher.unity.com).
2. Select the Asset Store package you want to deprecate.
3. Select **Deprecate**.
4. Select the **I agree to have the package deprecated** checkbox.

The [Status](AssetStoreStatus.html) of the package changes to **Deprecated**.
